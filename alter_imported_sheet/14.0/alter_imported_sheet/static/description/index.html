Import Data

This module provide feature to edit and save uploaded file(.csv & .xlsx) in odoo.

Click on Import Data menu.

Upload file for edit.

Click on load data button for load csv or xlsx data.

Hera you can see your uploaded file data.

Go to Setting -> Technical -> Attachments and see your uploaded file here.

Now here you can edit your file data and save file.

Download file.

Go to Download Folder and open Downloaded file.
